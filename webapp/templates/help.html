All responses are returned in JSON.

REQUEST: /
RESPONSE: The HP Wiki homepage (HTML).

REQUEST: /help
RESPONSE: This API reference.

ENDPOINT: /characters
Optional query parameters:
    house={HOUSE} case-insensitive substring match on a character's house  
    name={NAME}   case-insensitive substring match on a character's name  
    limit={LIMIT} maximum number of results (default: 100)

Response: A JSON array of character objects. 
            Each object has keys:
            id, name, house, gender, wand, patronus, hair_color

Example:
    GET /characters?house=gryffindor&limit=3

ENDPOINT: /spells
Optional query parameters:
    name={NAME}  substring match on spell name or incantation  
    type={TYPE}  substring match on spell type  
    effect={EFFECT} substring match on spell effect  
    light={LIGHT}   substring match on spell's light  
Response: A JSON array of spell objects.  
            Each object has keys:
            id, name, incantation, type, effect, light

Example:
    GET /spells?effect=force

        "effect": "Great Force",

ENDPOINT: /potions
Optional query parameters:
    name={NAME}       substring match on potion name  
    effect={EFFECT}   substring match on potion effect  
    difficulty={LEVEL}  substring match on difficulty level  

Response: A JSON array of potion objects. 
            Each object has keys:
            id, name, effect, difficulty_level, characteristics, ingredients

Example:
    GET /potions?effect=induce&difficulty=advanced
    [
        {
        "characteristics": "Sunshine yellow in colour, Sweet, Aromatic, Emits rainbow when finished",
        "difficulty_level": "Advanced",
        "effect": "Induces happiness",
        "id": 31,
        "ingredients": [
            "Shrivelfig",
            "Porcupine quills",
            "Peppermint sprigs",
            "Sopophorous beans",
            "Wormwood"
        ],
        "name": "Elixir to Induce Euphoria"
        }
  ]
